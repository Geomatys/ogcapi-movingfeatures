
[[clause-core-movingfeature]]
== Requirements Class "MovingFeatures"
=== Overview
include::requirements/movingfeature/requirements_class_movingfeature.adoc[]

The `MovingFeatures` requirements class defines the requirements for a moving feature.
A moving feature is an object that provides information about and access to <<resource-temporalGeometry-section,TemporalGeometry>> and <<resource-temporalProperties-section,TemporalProperties>>.
